spotify2tidal

  • JavaScript, HTML
  • PC, Mac, Linux

spotify2tidal is a simple application made to test out the Electron framework. The application allows you to drag a Spotify song link onto the interface to open the same song in Tidal. The purpose is to facilitate the use of Spotify's discovery features with a Tidal subscription.

KrokSynth

  • C++
  • PC, Mac

KrokSynth is a highly basic software synthesizer made using the iPlug framework. It features the basic wave shapes (sine, square, triangle, saw) and a basic ADSR envelope. The application is built as a VST3 for usage in DAWs such as Ableton Live and FL Studio.

ArrowQuest

  • Java
  • PC, Mac, Linux

ArrowQuest is a mod for the popular game Minecraft. It is a highly expansive mod which adds much to the exploration and combat aspects of the game. The mod also implements features commonly found in role playing games.

animation-strip-creator

  • Python
  • PC, Mac, Linux

animation-strip-creator is a simple application I made specifically to assist in the development of ArrowQuest (shown above). It takes a folder of equal-size images (each frame of an animation) and generates a vertical image strip. The purpose is to prepare custom 2D animations for usage in a video game or video game mod.
